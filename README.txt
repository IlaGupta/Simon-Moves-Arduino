This represents a game called Simon Moves.
Repository contains Circuit diagram of technical system, Code.
Hardware:
	Arduino Uno
	LCD
	Ball Mouse
Software:
	It contains the detailed code developed in Arduino Sketch Software with programming 	language C++
Game Rules:
	Observe the sequence of arrows displayed on LCD
	Use Mouse to follow same sequence as asked on screen
	Correct sequence win the game
	Incorrect gets the game over
